LLM integration utilities

Usage: ucp llm [OPTIONS] <COMMAND>

Commands:
  id-map       Create ID mapping for token efficiency
  shorten-ucl  Convert UCL to use short IDs
  expand-ucl   Convert UCL from short to full IDs
  prompt       Generate prompt documentation for UCL capabilities
  context      Manage context window for LLM
  help         Print this message or the help of the given subcommand(s)

Options:
  -v, --verbose          Enable verbose output
      --trace            Enable trace-level logging
  -f, --format <FORMAT>  Output format (text, json) [default: text] [possible values: text, json]
  -h, --help             Print help
  -V, --version          Print version
