UCL (Unified Content Language) operations

Usage: ucp ucl [OPTIONS] <COMMAND>

Commands:
  exec   Execute UCL commands
  parse  Parse and validate UCL without executing
  help   Print this message or the help of the given subcommand(s)

Options:
  -v, --verbose          Enable verbose output
      --trace            Enable trace-level logging
  -f, --format <FORMAT>  Output format (text, json) [default: text] [possible values: text, json]
  -h, --help             Print help
  -V, --version          Print version
